## jar 本地安装
```
# dubbo-2.8.4.jar
mvn install:install-file -Dfile=lib/dubbo-2.8.4.jar -DgroupId=com.alibaba -DartifactId=dubbo -Dversion=2.8.4 -Dpackaging=jar -DgeneratePom=true
mvn install:install-file -Dfile=lib/dubbo-2.8.4-sources.jar -DgroupId=com.alibaba -DartifactId=dubbo -Dversion=2.8.4 -Dpackaging=jar -DgeneratePom=true -Dclassifier=sources
# fastdfs-1.2.jar
mvn install:install-file -Dfile=lib/fastdfs-1.2.jar -DgroupId=org.csource.fastdfs -DartifactId=fastdfs -Dversion=1.2 -Dpackaging=jar -DgeneratePom=true
```
